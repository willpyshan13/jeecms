2019年四季度意见征集关于如何提升江州第七次全国人口普查工作实施的意见征集
征集时间： 2019-12-12 至 2019-12-25
状态： 进行中
来源： 福州市统计局

人口普查是一项严密的工作和数据质量的好坏与调查对象的信任和配合有着高度相关，为了有助于提高人口普查数据的质量和工作顺利开展，现针对如何提升普查对象对普查工作的信任感与配合度，诚邀您提出的意见建议，我们将根据收到的意见建议进行汇总整理，并研究运用在福州第七次全国人口普查工作中。谢谢您的参与！
