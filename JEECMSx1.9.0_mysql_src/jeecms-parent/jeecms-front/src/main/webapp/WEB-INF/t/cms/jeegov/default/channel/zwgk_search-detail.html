江州人民政府关于表彰2013-2015年度加快工业发展加速工业崛起先进单位的决定
日期：2019-11-16
来源：江州人民政府
分享：

各市、县（区）人民政府，省政府各部门：

2019年，在省委、省政府的正确领导下，全省各地认真贯彻落实党的十七届五中全会和全国教育工作会议精神，深入践行科学发展观，切实履行政府职责，坚持教育优先发展，有力保障教育投入，积极改善办学条件，转变教育发展方式，教育结构进一步优化，有力促进了各类教育的协调、均衡发展，为我省科学发展、跨越发展作出了新的成绩。

为总结经验，表彰先进，树立典型，推动工作，省政府决定，对在2010年度全省县级政府教育工作督导评估中获得优秀前三名的贵溪市、抚州市临川区、南昌市青云谱区、丰城市（其中：南昌市青云谱区、丰城市总分相同，并列第三名），授予“201o0年度江西省教育工作先进县（市、区）”荣誉称号，对其它达到优秀等次的永修县、上栗县、奉新县、婺源县、大余县、南昌市红谷滩新区、江西萍乡经济开发区、九江经济技术开发区、南昌经济技术开发区予以通报表扬。

希望受表彰、表扬的单位发扬成绩，再接再厉，为全省教育事业发展再创佳绩。各地、各有关部门要以先进单位为榜样，进一步加强对教育工作的领导，优化教育结构布局，提升教育现代化水平，为服务全省经济社会发展做出新的更大的贡献。
